What is Autodesk Forma?

Forma is Autodesk's cloud-based platform for end-to-end project delivery — from early-stage planning and conceptual design through to design coordination and construction. It's the rebranded successor to Autodesk Construction Cloud (ACC), so anything you used to do in ACC now lives here.

  • Document management — a single source of truth for drawings, models and project files, with full version control
  • Document reviews — markup, comment and approve documents in-browser without endless email chains
  • Working collaboratively — internal and external teams work on the same live information, in real time
  • Site data capture — capture progress, issues and observations on site and feed them straight back into the project

No separate account needed — Single Sign-On is enabled

Use your normal Aureos credentials to sign in. There's no extra password to remember and no separate Autodesk account to set up. If you've signed into your laptop this morning, you're already halfway there.

Working with Forma files? Install Desktop Connector

Forma itself runs in the browser, but if you'll be opening or syncing Forma project files locally (Revit, Civil 3D etc.), you'll need Autodesk Desktop Connector. Install it in one click via Company Portal.
